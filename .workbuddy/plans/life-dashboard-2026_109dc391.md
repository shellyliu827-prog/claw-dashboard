## 用户需求

在已有 `april-2026.html` 的基础上，构建完整的三层生活记录体系，三个页面互相跳转：

### 三层结构

```
index.html（2026 全年看板）
    └── q2-2026.html（Q2 季度视图）
            └── april-2026.html（四月日历，已有）
```

### 全年看板（index.html）

- 四个季度的主线目标卡片 + 完成状态标记（核心模块）
- 全年重要事件 / 纪念日时间轴（横向或纵向，标注已知事件）
- 个人年度数据统计区（跳舞次数、自媒体发帖数等，支持后续手动更新）
- 风格：比四月日历更简洁克制，减少装饰元素，偏正式浅色系

### Q2 季度视图（q2-2026.html）

- 7 条主线的季度进度条 + 每条主线的打卡记录列表（按时间倒序，当前均为空占位）
- 4 / 5 / 6 月三个月的日历入口卡片（4 月卡片可点击跳转，5/6 月标注"待建"）
- 风格：沿用四月日历的浅色纸质感

### 四月日历（april-2026.html）修改

- 在页面顶部加入导航面包屑：`2026 全年 › Q2 › 四月`，点击可跳转到对应页面

**Q2 主线（7条）：** 舞蹈（G社160次/2027.7.27）、阅读&写作、看牙、看耳朵、断舍离、财务、自媒体更新；另附游戏主线：死亡搁浅、燕云十六声、世界之外

**全年重要事件：** 4/15 柏源生日、4/26 朋友婚礼（安徽）

## 技术栈

纯 HTML + CSS，与现有 `april-2026.html` 保持一致，无 JavaScript 框架，无构建工具。字体沿用 `Inter + Noto Sans SC`（Google Fonts CDN）。

## 实现策略

三个独立 HTML 文件，通过 `<a href>` 相互链接。所有页面共享相同的 CSS 变量体系（颜色 Token 写在各自 `<style>` 内，保持一致即可），无需抽取公共 CSS 文件以降低维护成本。

### 颜色体系（三页统一）

| Token | 值 | 用途 |
| --- | --- | --- |
| 页面背景 | `#f5f3ef` | 米白纸质感 |
| 卡片背景 | `#ffffff` | 普通卡片 |
| 边框 | `#e8e4de` | 卡片/格子细线 |

| 强调色-绿 | `#7acf7a` | 假期 |


### 全年看板风格差异点

- 去掉背景装饰气泡，页面更纯净
- 标题用 Inter 500 weight（而非 700），更克制
- 季度卡片用 `border-left: 3px solid` 做色彩标识，而非彩色背景
- 时间轴用细线 + 圆点样式

## 设计方案

### 整体风格

三层页面统一使用浅色纸质感体系（`#f5f3ef` 米白底），Inter + Noto Sans SC 无衬线字体。全年看板比月度日历更克制简洁，减少装饰性元素，突出信息密度；Q2 页面和四月日历保持一致的轻手账感。

---

### 全年看板（index.html）— 简洁正式风

**Header 区**
大标题 "2026" 用 Inter 700 / 72px，深灰 `#1a1a1a`；副标题 "年度生活看板" 用 Inter 300 / 14px，字距拉宽，颜色 `#aaa`。无装饰气泡，纯净背景。

**数据统计行（Stats Bar）**
四个数据卡片横排：今年跳舞次数 / 自媒体发帖数 / 打卡天数 / 完成目标数。每卡白色背景，数字用 Inter 600 / 32px，标签 `#888` / 12px。

**季度目标区（核心）**
四行卡片（Q1–Q4），每行左侧有 3px 彩色竖线（Q1绿 / Q2金 / Q3蓝 / Q4紫），右侧是该季度主线目标列表（小 tag 形式），最右侧显示完成率进度环或进度条。Q2 行可点击跳转，其余季度标注"规划中"。

**全年时间轴**
横向时间轴，12个月份刻度，重要事件悬挂在对应月份下方，圆点 + 短竖线 + 事件文字。已知事件标注：4月（婚礼/生日），其余月份留空占位。

---

### Q2 季度视图（q2-2026.html）— 手账感

**面包屑导航**
顶部一行：`← 2026 全年 › Q2 Apr–Jun`，点击"← 2026 全年"跳回 index.html。

**主线进度区**
每条主线一行：图标 + 名称 + 进度条（0%） + 右侧打卡记录展开区。打卡记录以小 tag 形式横排（如 "4/8 第1次"），当前全部为空占位文字"暂无记录，快去打卡～"。

**月份入口卡片**
三张卡片横排（4月/5月/6月）：4月卡片有完整日历缩略图感（高亮今日格子），点击跳转 `april-2026.html`；5月/6月卡片灰色调，标注"待建"。
